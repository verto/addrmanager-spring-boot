A sample project using spring boot with angularjs and semantic-ui

Demo: [http://addrmanager-spring-boot.herokuapp.com](http://addrmanager-spring-boot.herokuapp.com)

## Libraries

- `Maven`
- `Spring boot`
- `Spring Data Jpa`
- `Spring MVC`
- `AngularJs`
- `Semantic-UI`

# Running 

```shell
mvn spring-boot:run
```

Access the project via: http://localhost:8080/
